# Argus: The Cognitive Shield

![Coverage](https://img.shields.io/badge/coverage-0%25-lightgrey)
![CI](https://github.com/priyansh-dimri/argus/actions/workflows/go.yml/badge.svg)

Argus is a Hybrid AI Web Application Firewall (WAF) written in Go. It solves the trade off between Latency and Context by merging the speed of deterministic pattern matching with the deep contextual understanding of probabilistic AI models.
